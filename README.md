# Curso de Linux Básico

Repositório dedicado aos materiais do curso de Linux Básico. O objetivo é transformar iniciantes em usuários confiantes no terminal.

## 🚀 Sobre o Aprendizado
Aprender Linux não é sobre decorar comandos, mas sobre entender a filosofia de um sistema onde "tudo é arquivo" e você tem o controle total sobre o hardware.
